# DCP Query Service documentation

This directory contains [Sphinx](https://www.sphinx-doc.org/) source files for the DCP Query Service docs. The
documentation system supports both Markdown and reStructuredText formats.

* To build the docs, run `make docs` in the root of the repository.
* To view the docs after building, run `open docs/_build/html/index.html`.
* Docs pushed to GitHub are automatically published on https://dcpquery.readthedocs.io/.
